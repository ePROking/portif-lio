Cookies e Web Storage

Cookies

Cookies são pequenos arquivos de texto armazenados no computador do usuário pelos navegadores da web. Eles são utilizados para lembrar informações sobre a visita do usuário, como preferências e autenticações. Os cookies podem ter data de expiração e são enviados ao servidor toda vez que uma página é carregada.

  • Tipos:
  • Cookies de Sessão: Expiram quando o navegador é fechado.
  • Cookies Persistentes: Permanecem até a data de expiração ou até serem removidos.

Web Storage

Web Storage é uma tecnologia que permite armazenar dados localmente no dispositivo do usuário. É dividida em duas partes: Local Storage e Session Storage. A tecnologia é ideal para criar experiências mais personalizadas em aplicativos da web.

  • Local Storage: Armazena dados sem data de expiração. Os dados persistem após o fechamento do navegador.
